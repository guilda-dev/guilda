# GUILDA: Grid & Utility Infrastructure Linkage Dynamics Analyzer
## About
GUILDAは
- 制御工学者から見て理解しやすく，使いやすい
- 電力システムや他分野の研究者の知見を融合しやすい

という特徴をもつことを目指した電力システムのシミュレータです．

## Requirement
- MATLAB
- Control system toolbox
- Optimization toolbox
